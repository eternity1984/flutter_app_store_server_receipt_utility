# flutter_app_store_server_receipt_utility

The receipt utility for the App Store Server API.

## Getting Started

```dart
void main() {
    const appReceipt = "...";
    final transactionId = ReceiptUtility.extractTransactionIdFromAppReceipt(appReceipt);
}
```
